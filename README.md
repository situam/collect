# collect

beginnings of collection

## installation

- download and extract [pocketbase](https://pocketbase.io/docs/) to `database/pocketbase
- install node v20:
  - [nvm](https://github.com/nvm-sh/nvm)
  - `$ nvm install 20`

## usage

$ cd database && ./pocketbase serve
$ cd frontend && npm run dev
